# Nokia Keypad

## Prerequisites

- ncurses
- xclip (to copy the output)

## Installation

Make sure you have all the prerequisites listed above.

Then simply run

```
make
```

## Keybinds

- `c` - Clear the output buffer.
- `y` - Copy the output buffer to clipboard.
- `#` - Toggle case sensitivity.
- `0` - Space
- `2-9` - Usual keypad binds.
- `q` - Quit the program.
